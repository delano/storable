Storable - v0.5

Marshal Ruby classes into and out of multiple formats (yaml, json, csv, tsv)

Installation

Via Rubygems, one of: 

$ sudo gem install storable
$ sudo gem install delano-storable --source http://gems.github.com/

or via download:

  • storable-latest.tar.gz
  • storable-latest.zip
